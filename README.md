# 🦽 Rolling Chaos

Rolling Chaos is an experimental browser-based 3D racing game where players take control of a wheelchair and race through an endless road while avoiding obstacles and mastering high-speed drifting.

Designed exclusively for desktop, the project combines arcade-style gameplay with modern web technologies to explore real-time 3D graphics and game mechanics directly in the browser.

> **Project Status:** 🚧 Under Development

---

## 🎮 Gameplay

* Control a high-speed wheelchair
* Drive through an endless road
* Avoid obstacles
* Drift around corners
* Activate speed boosts
* Aim for the highest score

---

## ✨ Current Features

* 🦽 3D wheelchair gameplay

## 🎯 Project Goal

Rolling Chaos began as a fun experimental project to learn browser-based game development using Three.js.

The long-term vision is to transform it into a polished arcade racing experience with engaging gameplay, customization, and progressively challenging levels.

## 🎮 Controls

| Key       | Action     |
| --------- | ---------- |
| **W / ↑** | Accelerate |
| **A / ←** | Turn Left  |
| **D / →** | Turn Right |
| **Space** | Brake      |
| **Shift** | Boost      |
